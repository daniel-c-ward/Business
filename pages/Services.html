Services

Website Desgin

This service is for clients who intend to build the website themselves or have it built by someone else , but want a professionally considered design to work from.

The focus is on layout, page structure, user flow and call-to-action placement. Design decisions are made with usablilty and clarity in mind, ensuring the site is easy to navigate and purpose driven. No development or coding is included in this service.

This service includes:

  • Page layout and structure
  • Navigation and content hirearchy
  • Call-to-action placement and user flow
  • Design guidance suitable for developer handoff or self-build

This this service does not includes:

  • Website development or coding
  • Hosting or deployment
  • Ongoing maintenance

Website Development

This service is for clients who want the website built for them, rather than just designed .

It covers the creation of a functional, responsive website based on an agreed structure and layout. Development focuses on clean implementation, usability, and reliability. Design decisions are applied as part of the build, but the outcome is a working website, not design files.

This service includes:

  • Building the website pages
  • Responsive layout implementation
  • Apply design and structure decisions
  • Testing across common screen sizes

This this service does not includes:

  • Ongoing maintenance unless agreed
  • Hosting unless added as a service
  • Large-scaled content creation

Website Rebuild

This service is for clients with an existing website that needs improvement, not just minor tweaks.

A rebuild takes the current site and restructures it, improving layout, usability, and implementation while preserving valuable content where appropriate. Both design and development work are involved, with the aim of delivering a noticeably better site.

This service includes:

  • Review of the existing website
  • Redesign of layout and structure
  • Rebuilding pages with improved implementation
  • Content reorganisation where necessary

This this service does not includes:

  • Ongoing maintenance unless agreed
  • Hosting unless added service
  • Full content rewriting
